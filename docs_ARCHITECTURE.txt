# Bologna Parking App — Architettura

## Obiettivo
App mobile per trovare parcheggio a Bologna in tempo reale,
con predizioni basate su dati storici.

## Zona pilota
Saragozza, Murri, zona Ospedale Sant'Orsola

## Stack
- Backend: Python + FastAPI
- Frontend: React Native + MapLibre
- Database: PostGIS (PostgreSQL)
- Dati live: SostaBo API + TPER
- ML: Prophet (predizioni orarie)
- Deploy: Railway
